# 设置项

打开方式：右上角齿轮图标 / `⌘ + ,`

## LLM Provider

配置 AI 助手使用的语言模型。

| 字段 | 说明 |
|------|------|
| 预设方案 | OpenAI / DeepSeek / 智谱 / Moonshot / 通义千问 |
| Base URL | 自定义端点（兼容 OpenAI 协议即可） |
| API Key | 必填，存储在本地 SQLite |
| Model | 如 `gpt-4o-mini` / `deepseek-chat` |

**测试连通性**按钮会调用模型让它回复 "OK"，验证 Key 有效。

## 索引管理

| 操作 | 说明 |
|------|------|
| 开始索引 | 启动后台扫描 |
| 暂停 / 恢复 | 临时停止/继续索引 |
| 取消 | 完全停止当前索引任务 |
| 已索引目录 | 显示已扫描的根目录列表 |

## 统计

显示当前索引的：
- 文件总数
- 分块总数
- 标签总数

## 主题

- **浅色** — 强制浅色
- **深色** — 强制深色
- **跟随系统** — 跟随 macOS appearance

切换实时生效。

## 语言

- 简体中文
- English

切换后立即应用，配置持久化。

## 忽略规则

每行一个 glob 模式：

```
*.log
*.tmp
~/.npm/
.vscode/
```

保存后下次索引生效。已索引的文件不会被回溯清理，需要取消索引 → 重新开始。

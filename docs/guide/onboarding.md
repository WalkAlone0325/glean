# 首次启动

Glean 启动后会自动检测是否为首次运行，并显示引导向导。

## 欢迎页

三块功能介绍：
- 🔍 **语义检索** — 用自然语言找文件
- 🤖 **AI 助手** — 对话式管理文件
- 🔒 **隐私优先** — 全部本地运行

点击「下一步」继续。

## 选择索引文件夹

默认会扫描三个目录：

- `~/Documents`
- `~/Downloads`
- `~/Desktop`

你可以：
- 点击「+ 添加文件夹」选择其他目录
- 点击每行的「✕」移除该目录

::: tip
建议首次只索引重要目录，扫描更快。后续可在设置中随时添加。
:::

## 开始索引

点击「开始索引」后，Glean 会在后台扫描所选目录：

1. **遍历文件**（智能跳过 .git/node_modules 等）
2. **提取元数据**（大小、修改时间、类型）
3. **生成全文索引**（SQLite FTS5）
4. **生成向量嵌入**（用于语义搜索，可暂停）

索引期间你可以正常使用应用，搜索结果会随着索引进度实时更新。

## 跳过向导

如果你点击「跳过」，Glean 不会索引任何文件夹，但应用可以正常打开。要开始索引，前往：

**设置 → 索引管理 → 开始索引**

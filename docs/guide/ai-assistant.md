# AI 助手

Glean 的 AI 助手不仅能回答问题，还能**执行任务**。

## 对话

按 `⌘B` 打开右侧聊天面板，输入问题即可开始对话。

支持的特性：

- 流式响应（边生成边显示）
- Markdown 渲染（代码高亮 / 表格 / 列表）
- 引用文件（点击可打开或在 Finder 中显示）
- 多轮上下文
- 对话历史侧边栏

## Agent 工具调用

AI 助手内置 5 个工具：

| 工具 | 说明 | 是否需要确认 |
|------|------|--------------|
| `search_files` | 全文检索 | 否 |
| `read_file` | 读取文件内容预览（≤32KB） | 否 |
| `list_similar` | 找相似文件 | 否 |
| `move_file` | 移动/重命名文件 | **是** |
| `tag_file` | 给文件打标签 | **是** |

### 工作流程

1. 你提问 → AI 决定调用哪些工具
2. 工具调用实时显示在助手气泡中（可展开查看参数/结果）
3. **写操作**会弹出确认对话框，可拒绝
4. 工具结果回喂给 AI → 继续推理或返回最终回答

### 撤销

每个写操作都记录在 `operations` 表中，工具卡片上的「撤销」按钮可一键回退：

- **移动文件**：反向 rename 回原路径（注意：覆盖操作不可撤销）
- **打标签**：移除新加的 file_tags 关联

## RAG 模式

聊天面板顶部有 `RAG` 复选框：

- **开启**（默认）：你的问题会先做语义检索，把相关 chunks 作为上下文喂给 LLM
- **关闭**：纯对话，不检索本地文件

引用的文件会列在助手回答下方，点击「打开」直接查看原文件。

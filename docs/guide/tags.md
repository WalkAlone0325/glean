# 标签与收藏

## 标签

每个文件可以打多个标签，便于分类与检索。

### 创建标签

**方式一**：在文件详情面板

1. 选中文件 → 右侧 DetailPanel
2. 在「标签」区域点击 + 按钮
3. 输入标签名 + 选颜色 → 添加

**方式二**：通过 AI 助手

```
给 ~/Documents/report.pdf 打上 "财务" 和 "2024" 标签
```

AI 会调用 `tag_file` 工具，需用户确认。

### 标签颜色

8 种预设颜色：红、橙、黄、绿、蓝、紫、粉、灰。

### 管理标签

设置页可以查看所有标签 + 文件数（管理功能开发中）。

## 收藏（星标）

文件可以加入「星标」便于快速访问。

### 添加星标

- 文件列表每行右侧的星形按钮
- AI 助手（暂未提供工具，仅 UI 操作）

### 查看星标

- 左侧边栏 → 「星标文件」
- 或文件列表顶部 → 「星标文件」tab

## 最近查看

Glean 自动记录你点击过的文件，按时间倒序展示。

- 最多保留 50 条
- 仅记录点击行为，不记录阅读时长
- 数据存本地，不上传

# 快捷键

## 全局快捷键

| 快捷键 | 功能 | 备注 |
|--------|------|------|
| `⌘ + ⇧ + Space` | 显示 / 隐藏主窗口 | 系统级，任何应用下都可用 |

::: tip
与 macOS Spotlight（`⌘ + Space`）不冲突。
:::

## 应用内快捷键

| 快捷键 | 功能 |
|--------|------|
| `⌘ + K` | 打开文件搜索面板 |
| `⌘ + B` | 切换 AI 助手面板 |
| `⌘ + ,` | 打开设置 |
| `⌘ + N` | 新建对话（在 AI 面板内） |
| `Enter` | 打开选中文件 / 发送消息 |
| `Esc` | 关闭面板/弹窗 |
| `↑ ↓` | 在列表中移动选择 |
| `Tab` | 搜索面板中切换模式 |

## AI 助手面板内

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 发送消息 |
| `⇧ + Enter` | 换行 |
| `⌘ + Enter` | 强制发送（在 textarea 中） |

## 文件列表右键菜单

右键单击文件行打开：

- 打开
- 在 Finder 中显示
- 复制路径
- 添加/移除星标

## 自定义快捷键

暂不支持自定义。如需此功能，请 +1 [Issue #暂未创建]。

## 与 macOS 系统冲突？

如果某个快捷键被系统占用，可在 **系统设置 → 键盘 → 键盘快捷键** 中关闭对应系统功能。
